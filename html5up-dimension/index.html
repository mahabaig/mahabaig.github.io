Overview

Welcome to my portfolio! I’m a Security+ certified cybersecurity professional with hands-on experience in Microsoft Sentinel, Azure SOC labs, and threat analysis. I specialize in identifying and responding to network anomalies, building dashboards for visibility, and documenting incident findings clearly for teams and auditors.

Scroll down to explore my projects in cloud security and threat monitoring — including my Azure SOC Lab and VPC Traffic Analysis project.

Projects

Azure SOC Lab (2025) – Created a honeypot virtual machine in Azure and configured Microsoft Sentinel for real-time monitoring. Captured and analyzed over 30,000 security events, identified brute-force patterns, and visualized global attack origins through KQL-based dashboards.

VPC Traffic Flow & Security Project – Configured AWS VPC flow logs to analyze network segmentation, validated inbound/outbound rules, and enforced least-privilege access between subnets. Used log analytics to detect misconfigurations and improve visibility.

Cloud Security with IAM Roles – Implemented AWS IAM roles and policies to test identity-based access control. Validated policy boundaries and prevented privilege escalation by enforcing least privilege across simulated user roles.
